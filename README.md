# Micro SaaS Scout - 出海工具需求挖掘系统

一个自动化挖掘出海Micro SaaS工具需求的平台，帮助发现下一个赚钱的小工具。

## 功能特点

### 🎯 核心功能
- **自动化需求挖掘**：从多个平台自动收集工具需求
- **智能评分系统**：从多个维度评估需求潜力
- **用户画像分析**：分析目标用户特征和使用场景
- **市场数据展示**：展示市场规模、增长趋势等关键数据
- **技术可行性评估**：评估开发复杂度和技术栈

### 📊 数据维度
1. **需求强度** - 问题紧迫性和讨论热度
2. **市场规模** - 搜索量、竞品用户数、增长率
3. **付费意愿** - 价格讨论和付费信号
4. **技术可行性** - 开发复杂度、时间、技术栈
5. **被动收入适配度** - 自动化程度和客户留存

### 🎨 界面展示
- **仪表板**：关键指标和趋势图表
- **需求卡片**：详细展示每个需求的完整信息
- **筛选系统**：按评分、类型、价格等快速筛选
- **数据可视化**：图表展示趋势和分布

## 技术栈

### 前端
- **Next.js 14** - React框架，SSR支持
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Recharts** - 数据可视化
- **Lucide React** - 图标库

### 部署
- **Vercel** - 推荐部署平台（自动部署、全球CDN）
- 也可部署到Netlify、Cloudflare Pages等

## 快速开始

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 部署到Vercel
1. 将代码推送到GitHub仓库
2. 访问 [Vercel](https://vercel.com)
3. 导入GitHub仓库
4. 自动部署完成，获得生产URL
